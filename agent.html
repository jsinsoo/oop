1. 청소 로봇 에이전트

핵심 설명
상태: 위치, 배터리, 먼지 여부 / 행동: 감지하기, 청소하기, 이동하기
class CleaningAgent:
    def __init__(self):
        self.location = "거실"
        self.battery = 100
        self.dust = True

    def sense(self):
        return self.dust

    def clean(self):
        if self.sense():
            self.dust = False
            self.battery -= 10
            return "청소 완료"
        return "먼지 없음"

    def move(self, place):
        self.location = place
        self.battery -= 5

agent = CleaningAgent()
print(agent.clean())
agent.move("주방")
print(agent.location, agent.battery)

상세 설명

이 예시는 지능 에이전트를 클래스로 표현한 것입니다. self.변수는 에이전트가 현재 가지고 있는 상태이고, 클래스 안의 함수인 메서드는 감지, 판단, 이동, 추천, 제어 같은 행동을 수행합니다.

문항 1

CleaningAgent 클래스에서 battery는 무엇을 표현하는가?

2. 온도 조절 에이전트

핵심 설명
상태: 현재 온도, 목표 온도 / 행동: 온도 비교, 냉방 또는 난방 결정
class TemperatureAgent:
    def __init__(self, current, target):
        self.current = current
        self.target = target

    def decide(self):
        if self.current > self.target:
            return "냉방 실행"
        elif self.current < self.target:
            return "난방 실행"
        return "유지"

agent = TemperatureAgent(30, 24)
print(agent.decide())

상세 설명

이 예시는 지능 에이전트를 클래스로 표현한 것입니다. self.변수는 에이전트가 현재 가지고 있는 상태이고, 클래스 안의 함수인 메서드는 감지, 판단, 이동, 추천, 제어 같은 행동을 수행합니다.

문항 2

현재 온도가 목표 온도보다 높으면 출력되는 행동은?

6. 스마트 조명 에이전트

핵심 설명
상태: 밝기, 사람 감지 여부 / 행동: 조명 켜기 또는 끄기
class LightAgent:
    def __init__(self, brightness, human):
        self.brightness = brightness
        self.human = human

    def control(self):
        if self.human and self.brightness < 40:
            return "조명 켜기"
        return "조명 끄기"

agent = LightAgent(20, True)
print(agent.control())

8. 알림 에이전트

핵심 설명
상태: 일정 목록 / 행동: 일정 추가, 일정 알림
class ReminderAgent:
    def __init__(self):
        self.tasks = []

    def add_task(self, task):
        self.tasks.append(task)

    def remind(self):
        return self.tasks

agent = ReminderAgent()

10. PEAS 기반 택배 로봇 에이전트

핵심 설명
상태: 성능점수, 환경, 액추에이터, 센서 / 행동: 배달 수행
class DeliveryAgent:
    def __init__(self):
        self.performance = 0
        self.environment = "복도"
        self.actuator = "바퀴"
        self.sensor = "카메라"

    def deliver(self, success):
        if success:
            self.performance += 10
            return "배달 성공"
        self.performance -= 5
        return "배달 실패"

agent = DeliveryAgent()
print(agent.deliver(True))
print(agent.performance)

상세 설명

이 예시는 지능 에이전트를 클래스로 표현한 것입니다. self.변수는 에이전트가 현재 가지고 있는 상태이고, 클래스 안의 함수인 메서드는 감지, 판단, 이동, 추천, 제어 같은 행동을 수행합니다.
